# 🔐 Password Strength Checker

## 📖 Description

This Python project checks the strength of a password and gives suggestions to improve it.

## ✨ Features

- Checks password length
- Detects uppercase letters
- Detects lowercase letters
- Detects numbers
- Detects special characters
- Calculates password score
- Shows password strength
- Gives improvement suggestions

## ▶️ How to Run

```bash
python password_checker.py
```

## 💻 Example

```
Enter Password: Hello@123

Password Score: 100/100

✅ Excellent Password
```

## 🛠 Built With

- Python 3
